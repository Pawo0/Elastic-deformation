# Finite element method (FEM) for elastic deformation

## Usage
```bash
git clone https://github.com/Pawo0/Elastic-deformation.git
cd Elastic-deformation
npm install 
npm run dev 
npm run server
```
open browser and go to http://localhost:5173/
